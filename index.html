Guía de Conexión MJPEG (TouchDesigner):

Para que esta aplicación reciba un stream en tiempo real, debes configurar TouchDesigner para exponer el stream como una URL web accesible.

  1. En TD: Utiliza un operador 'Movie File In' o 'Movie Stream'.
  2. Configuración del Stream: Asegúrate de que la fuente de video esté configurada correctamente (ej. cámara, archivo).
  3. Exponer URL: El método más sencillo es usar el nodo **Movie Stream** y obtener su URL de salida. Esta URL suele tener un formato como `http://localhost:8080/stream.mjpeg`.
  4. En esta App: Pega esa URL en el campo superior y haz clic en 'Conectar'. La aplicación cargará el stream automáticamente.
